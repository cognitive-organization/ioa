### DEC-2026-03-02-006

- **Date:** 2026-03-02
- **Project:** CREAOS (paribali-labs/creator-copilot)
- **Decision:** Move Prisma CLI and @prisma/client to root devDependencies and run generate via the direct binary in Docker builds to resolve version resolution conflicts
- **Context:** Prisma auto-installs a matching @prisma/client during `generate`, which broke `--frozen-lockfile` Docker builds; hoisting both packages to root ensures the binary finds its client, and running the binary directly bypasses pnpm workspace hoisting edge cases.
- **Alternatives:** Use `pnpm exec prisma generate` per workspace package, disable Prisma auto-install, use npx (downloads latest, wrong version)
