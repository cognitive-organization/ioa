### DEC-2026-03-01-002

- **Date:** 2026-03-01
- **Project:** IOA (cognitive-organization/ioa)
- **Decision:** Switch all holding workflow Slack delivery scripts from bash variable interpolation inside Python one-liners to Python-only file-read with a static fallback
- **Context:** Embedding `$REPORT` bash variables directly in Python one-liners caused SyntaxError when report content contained newlines or quote characters; the Python file-read approach is shell-injection-safe and handles arbitrary content.
- **Alternatives:** Escape/sanitize bash variables before interpolation, use a heredoc, switch to jq-based JSON payload construction
- **Made by:** Human (Paulo Lima) + Agent (Claude Opus 4.6)
- **Source:** commit 7a069c58 (cognitive-organization/ioa)
- **Review date:** 2026-03-31
